# 幼儿园教师端小程序 — 信息架构文档

## 概述

- **产品名称**: 幼儿园教师端小程序
- **页面总数**: 5 个主 Tab 页面，约 50+ 个子页面
- **最大深度**: 4 层
- **IA 框架**: Organization / Labeling / Navigation / Search

---

## 一、组织系统 (Organization System)

### 1.1 组织方案：Category（按类别分组）

采用 **LATCH** 中的 **Category（类别）** 方案，以教师工作场景为分组依据：

| Tab | 组织逻辑 | 适用场景 |
|-----|---------|---------|
| 首页 | 高频入口聚合 | 快速触达待办、通知、常用功能 |
| 资源中心 | 内容型资源沉淀 | 查阅资源库与案例库 |
| 园务中心 | 管理型文件分发 | 查阅园所管理文件 |
| 成长中心 | 个人型成长档案 | 师德师风、培训记录、共建任务、个人成果 |
| 家园共育 | 家园协同管理 | 上传在园照片、发布亲子任务、管理儿童成长档案、完成度汇总 |

### 1.2 分类体系：层级式 (Hierarchical)

- **首页**: 信息聚合 → 快速分流
- **资源中心**: 资源库 + 案例库双入口 → 八乡筛选（资源库）/ 按年级领域筛选（案例库），资源与案例相互链接
- **园务中心**: 文件分类入口 + 最近上传 → 按文件类型分类（党建/政策/制度/后勤）
- **成长中心**: 个人中心 → 按成长维度细分（师德/培训/共建/档案）
- **家园共育**: 家园协同 → 在园时光、亲子任务、儿童成长档案、完成度汇总

### 1.3 内容类型定义

```yaml
content_types:
  - 课程资源:
      属性: [标题, 适用年龄段, 领域, 活动类型, 附件]
  - 活动案例:
      属性: [标题, 主题, 年龄段, 活动目标, 材料准备, 活动流程, 借鉴点]
  - 课题资料:
      属性: [课题名称, 负责人, 阶段, 材料类型, 附件]
  - 幼儿成长档案:
      属性: [幼儿姓名, 班级, 学期, 五大领域数据, 教师建议]
  - 通知公告:
      属性: [标题, 类型, 发布时间, 内容, 附件]
  - 培训资料:
      属性: [标题, 培训类型, 时间, 资料文件, 学习记录]
```

## 三、导航系统 (Navigation System)

### 3.1 全局导航：底部 Tab Bar

```
┌──────────┬──────────┬──────────┬──────────┬──────────┐
│   首页   │   资源   │   园务   │   成长   │   家园   │
└──────────┴──────────┴──────────┴──────────┴──────────┘
```

- **模式**: 微信小程序底部 Tab Bar（固定 5 个入口）
- **始终可见**: 所有页面底部均显示 Tab Bar
- **状态保持**: 切换 Tab 不丢失页面状态

### 3.2 局部导航

**首页 — 垂直滚动 + 模块卡片**:
- 顶部轮播（自动/手动滑动）
- 待办事项（横向 Tab 切换：全部 / 资源审核 / 共建任务 / 活动报名，含状态筛选）
- 园所通知（纵向列表，按类型分组：活动通知 / 作品征集 / 放假安排，含状态筛选）
- 常用入口（Icon 网格布局，4 列）
- 推荐案例（横向滑动卡片）

**资源中心 — 顶部搜索 + 双入口**:
- 顶部搜索栏（固定）：关键词搜索 / 搜索历史 / 结果筛选
- 资源库：八乡筛选（乡味|乡音|乡情|乡貌|乡趣|乡美|乡源|乡贤）→ 资源卡片列表 → 资源详情页（简介|标签|获取|转化|课程应用）
- 案例库：按年级 / 按领域筛选 → 案例卡片 → 案例详情页（简介|转化|资源链接|详案下载）
- 资源详情页与案例详情页相互链接

**园务中心 — 分类文件入口**:
- 快捷入口：党建学习 / 政策学习 / 园所制度 / 后勤保障 / 最近更新
- 最近上传文件列表（预览 / 下载）

**成长中心 — 个人成长工具**:
- 师德师风：相关文件（预览/下载）/ 案例学习
- 培训学习：园内培训 / 外出培训 / 心得上传
- 共建任务：查看任务 / 提交材料 / 查看审核
- 教师成长档案：个人信息 / 证书成果 / 获奖记录 / 公开课

**家园共育 — 家园协同**:
- 在园时光：发布活动（观察内容：儿童|地点|实录|分析|图片 + 可见性设置）/ 过往活动（时间轴）/ 进度表
- 亲子任务：发布任务（教师填写背景|要求）/ 任务列表（整体进度 + 个人进度：未读|已读|已提交）
- 儿童成长档案：发布家长评价（月度|学期）/ 教师月度评价（引用过往照片）/ 教师学期评价（五大领域|身高体重|量表|总结）/ 评价进度总表
- 完成度汇总：按幼儿姓名，四列展示在园时光 | 亲子任务 | 儿童成长档案的完成状态

### 3.3 跨页面跳转关系

| 源页面 | 目标页面 | 触发方式 |
|--------|---------|---------|
| 首页 → 待办·资源审核 | 园务中心 | 点击待办卡片 |
| 首页 → 待办·共建任务 | 成长中心 → 共建任务 | 点击待办卡片 |
| 首页 → 常用·教案资源 | 资源中心 | 点击入口 Icon |
| 首页 → 常用·课程管理 | 资源中心 | 点击入口 Icon |
| 首页 → 常用·家园沟通 | 家园共育 | 点击入口 Icon |
| 首页 → 推荐案例 | 资源中心 → 案例详情 | 点击案例卡片 |
| 资源中心 → 案例详情 | 案例详情页 | 点击案例 |
| 资源中心 → 资源简介 | 资源简介页 | 点击资源名称 |
| 成长中心 → 共建任务 | 园务中心（审核端）| 提交流程 |

### 3.4 导航深度规范

| 层级 | 示例路径 | 最大项目数 |
|------|---------|-----------|
| Level 1 | Tab Bar | 5 个 |

| Level 3 | 子模块/列表 | 5-8 个 |

## 四、搜索系统 (Search System)

### 4.1 搜索入口

- **位置**: 资源中心顶部（固定搜索栏）
- **搜索范围**: 主题资源、活动案例
- **提示文案**: "搜索主题资源、活动案例"

### 4.2 搜索组件

| 组件 | 设计 |
|------|------|
| 搜索框 | 顶部固定，带 placeholder 提示 |
| 搜索历史 | 记录最近搜索关键词 |
| 结果筛选 | 按内容类型筛选（主题资源 / 活动案例） |
| 结果排序 | 按相关度 / 时间 / 浏览量 |
| 无结果 | 显示"未找到相关内容，试试其他关键词" |

### 4.3 筛选维度（Faceted）

```yaml
facets:
  - 内容类型: [主题资源, 活动案例]
  - 适用年龄段: [小班, 中班, 大班]
  - 活动领域: [语言, 艺术, 健康, 科学, 社会]
  - 发布时间: [本周, 本月, 本学期, 自定义]
```

## 七、关键设计决策

1. **首页定位为"工作台"而非"内容页"**: 聚合待办、通知、快捷入口，降低教师操作路径
2. **资源中心双入口设计**: 资源库（按主题浏览）+ 案例库（按年级/领域浏览），同一套数据不同视角
3. **园务中心为文件分发平台**: 按类型分类的快捷入口 + 最近上传，简化教师查找文件路径
4. **成长中心聚焦教师个人发展**: 师德师风、培训学习、共建任务、成长档案四位一体
5. **家园共育独立为第五 Tab**: 将家园沟通、亲子任务、儿童成长档案整合为协同工作空间
6. **跨页面数据复用**: 首页的待办事项与其他 Tab 共享数据源，减少数据孤岛
